#### The features of the app involves:
1. The app tracks the spends for the day and for the whole month.
2. The app has the feature to add an expense with voice input. It can parse the amount and the amount spent on automatically.
3. The app automatically tags the expense to a specific category and we can get an understanding of the spends category-wise.
4. The app shows all the expense entries made in the reverse chronological order.
5. The app contains the analytics screen where we can visualize the spends throgh pie chart and a bar graph showing the data for the current month.
6. The app has a functionality to export the expenses based on date range filter to an exce file. And the file gets saved to the Downloads folder in the device.
7. The app has a feature to auto-detect the expenses made digitally (NEFT/UPI/Card Payments) where an SMS is received from the bank, and add that automatically to the app.
